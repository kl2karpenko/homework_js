## 5 задание

Переписать if'ы в switch
важность: 4решение
Перепишите код с использованием одной конструкции switch:

```
var a = +prompt('a?', '');

if (a == 0) {
  alert( 0 );
}
if (a == 1) {
  alert( 1 );
}

if (a == 2 || a == 3) {
  alert( '2,3' );
}
```
